
### 1. Install Dependencies

```bash
$ git clone https://github.com/AlexandrDegteari/giftest.git
$ cd giftest
$ npm install
```

### 2. Build Instructions

```bash
$ npm run build
```

### 3. Run Instructions

Run the following (default web browser will open to `http://localhost:8080/`):

```bash
$ npm start
```
